### Avail Cross-Chain USDC Integration Overview

- Cross-chain swap functionality using unified balances

- Aggregates USDC/stables across multiple chains into single spendable balance
- Users can spend up to total asset value without manual bridging
- One extra transaction minimum, multiple if setting individual allowances
- SDK handles parallel collection from source chains
- Demo showed $22 worth of tokens (USDC, ETH, USDT) across chains

- Can specify output token needed on destination chain
- Supports any token conversion, not just stables
- Live on Monad testnet with USDC/USDT/ETH support

- Wrapped ETH coming soon
- Uses other bridge protocols (Relay, etc.) as infrastructure layer

### TimeBlock Integration Potential

- Perfect fit for options trading with cross-chain premium payments

- Users can pay option premiums from combined balances across chains
- Removes friction of bridging before trading
- Reverse flow (profit distribution) would require separate implementation

- Contract logic needed to split payouts across multiple chains
- Additional SDK requests for withdrawal distribution
- Avail positioned as input layer, not swap product

- Focuses on aggregating assets for app usage vs. token swapping
