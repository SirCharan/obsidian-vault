### UI Design Review - Markets Page

- Cards currently too large for optimal layout

- Reduce card size by ~33% to fit 4 cards per line
- Alternative: 3 cards per line with reduced size acceptable
- Font hierarchy issues identified

- Token names should be largest text element
- “Graduated Valajo toolbar” font too small, needs increase
- Cap size numbers too large relative to token names
- Remove “Top Movers” section from top right

- No clear purpose identified
- Improve scroll indication

- Add second row of cards if possible
- Make list below cards visible on initial screen
- Enhance scrolling intuitiveness

### Text Sizing Inconsistencies

- All section titles too small throughout interface

- “All chains”, “Advanced filters”, other headings need enlargement
- Current 15-16px font size inadequate for titles
- Size disparity between card elements and page headers

- Large token names in cards vs small section headings creates visual imbalance
- Need middle ground approach: increase headers or reduce card text (or both)
- Reference comparison to Pump.fun

- Their consistent small sizing works because everything proportional
- Current design lacks visual hierarchy consistency

### Launch Token Page Feedback

- Overall design approved with minimal changes needed
- Add border around launch panel

- Currently appears floating without clear boundaries
- Text visibility issues

- Grayscale/white text not 100% visible against background
- Consider contrast improvements
- Page vs modal decision confirmed

- Separate page approach preferred over modal
- Maintains user attention on launch process

### Next Steps

- Implement 4-card layout with reduced card sizes
- Establish consistent UI font size rules
- Prioritize markets page improvements (primary user entry point)
- Timeline: Few hours for initial updates
