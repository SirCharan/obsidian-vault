### Points Program Design & Structure

- Building comprehensive community engagement system for DeFi platform
- Integration with Discord for leveling up through discussions
- Daily prediction game bot

- ETH price predictions with 2-hour windows
- Points awarded based on accuracy
- Simple missions for ecosystem onboarding

- Connect wallet, follow on X, join Discord, share expo file
- Reference: GitCoin’s onboarding model with protocol partnerships
- Daily streak system with UI checkbox showing activation status

- Extra points for maintaining streaks (10-day, 15-day)
- Shield protection feature for missed days (earned via Kapoor ball)

### Gamification Elements & Rewards

- “Kapoor ball” instead of traditional spin wheel

- Mystical theme alignment vs common slot machines
- Rewards include streak shields and point multipliers
- Point multipliers for actual product usage (mainnet/testnet launch)
- Kapoor Points as burnable currency for premium passes
- Strategy shift: community engagement → product usage incentivization
- Reference model: Hyper Liquid’s pool-specific incentive controls

### Technical Implementation & Design

- Separate landing page for points program (not main site)
- Mobile-first design priority for community accessibility
- One feature per screen with scroll navigation
- Matrix + retro 1890s PC arcade aesthetic
- No brand matching requirements except logo
- Design freedom for fonts, colors, experimental elements
